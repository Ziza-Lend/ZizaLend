# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the ZizaLend platform.

## What is an ADR?

An Architecture Decision Record (ADR) is a short document that captures an important architectural decision made along with its context and consequences.

## Template

```markdown
# ADR-NNN: Title

## Status

[Proposed | Accepted | Deprecated | Superseded]

## Context

What is the issue that motivated this decision?

## Decision

What is the change being proposed?

## Consequences

What becomes easier or more difficult?
```
